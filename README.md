# PocketPad — 未来ガジェット013号

> **Your PC. In Your Pocket.**
> スマホがPCのトラックパッド・キーボード・リモコンになるアプリ。
> ゲーム中・ソファ・布団の中から、PCを瞬時に操作。

## 🎮 これは何？

Androidスマホから、同じWiFi内のWindows PCを操作できるアプリです。

- 📱 **トラックパッド** — スマホの画面をなぞるとPCのマウスが動く
- ⌨️ **日本語入力** — スマホで文章を書いてEnter一発でPCに送信（スマホのIMEで変換できるので速い）
- 🖱️ **クリック・スクロール** — タップ=左クリック / 長押し=右クリック / 右端の帯で上下スクロール
- 🎛️ **マクロボタン** — Stream Deck風のボタングリッド。コピペ・アプリ起動・スクショ転送・
  音量などのメディアキー・スリープまでワンタップ。**PCのブラウザから約50種のプリセットを選んで自由にカスタマイズ**
- ▶️ **YouTubeコントローラー** — 再生/一時停止・10秒送り戻し・音量・全画面を専用ページで
- 📸 **スクショ転送** — PCの画面を撮ってスマホでプレビュー
- ⚙️ **レイアウト設定** — ページの表示/並び順・下部ボタン・トラックパッド感度を変更可能。
  設定はPCと自動同期

## 🧰 必要なもの

| 機材 | 条件 |
|------|------|
| PC | Windows 10 / 11（64bit） |
| スマホ | Android 8 以上 |
| ネットワーク | PCとスマホが**同じWiFi**につながっていること |

インストール不要のもの：.NETランタイム（EXEに同梱）／サーバー／アカウント登録。**すべて無料・LAN内で完結、クラウドに何も送りません。**

## 🚀 かんたんセットアップ

### ① PC側（1分）

1. [Releases](../../releases) から `PocketPadTray.exe` をダウンロード
   （まだReleaseがない場合は下の「開発者向け」の手順でビルドしてください）
2. ダブルクリックで起動
3. Windowsファイアウォールのダイアログが出たら「**プライベートネットワーク**」にチェックして許可
4. 画面右下のタスクトレイに常駐します。アイコンを**左クリックすると接続QRコード**が表示されます

> 💡 トレイアイコンを右クリック →「**Windows起動時に自動起動**」にチェックを入れると、PC起動時に自動で立ち上がります（もう一度クリックで解除）。

### ② スマホ側（1分）

1. [Releases](../../releases) から `app-release.apk` をダウンロードしてインストール
   （「提供元不明のアプリ」の許可が必要です）
2. アプリを開いて「**QRコードで接続**」→ PC画面のQRを読み取るだけ
   （手動でIPアドレスとトークンを入力してもOK。トレイ右クリック→「接続情報を表示」で確認できます）
3. トラックパッド画面になれば成功！

> 💡 2回目からは接続情報が保存されているので、アプリを開くと自動で再接続します。

## 📖 使い方

| 操作 | 動作 |
|------|------|
| 画面をなぞる | マウスカーソル移動 |
| タップ | 左クリック |
| ダブルタップ | ダブルクリック |
| 長押し | 右クリック |
| タップ後すぐなぞる | 掴んで移動（ドラッグ） |
| 右端の帯を上下 | スクロール |
| 切替バー（タップ/スワイプ） | トラックパッド ⇄ マクロ ⇄ YouTube のページ切替 |
| 下部ボタン | Enter / ⌫バックスペース（長押しで連続削除）/ キーボード / Alt+Tab / Win |

**テキスト入力：** キーボードボタン → 入力欄に文章を書いて **Enterで一気にPCへ**（PC側のEnterも自動で押されます）。スマホの日本語変換がそのまま使えるので、PCの前にいなくても長文が快適に打てます。

## 🎛️ カスタマイズ（設定ダッシュボード）

**PC側のトレイアイコンを右クリック →「設定ダッシュボードを開く」** でブラウザから全設定を編集できます。

- **マクロボタン** — 約50種のプリセット（コピペ・ウィンドウ操作・アプリ起動・ブラウザ操作・
  スクショ・メディアキー・電源操作・マクロ手順）をクリックで追加。ラベル・アイコン・色・
  動作も自由に作成/編集/並び替え
- **ページ / 下部ボタン** — 使うものだけ表示・並び替え
- **トラックパッド感度** — スライダーで調整
- 変更は「保存」を押すと**接続中のスマホに即反映**。スマホ側の設定画面（切替バー右端の⚙）
  からも同じ内容を編集でき、どちらで変えても自動で同期されます
- ダッシュボードは localhost 限定公開（LAN内の他端末からはアクセス不可）

> ⚠️ シャットダウン等の電源ボタンは、スマホ側で確認ダイアログを挟んでから実行されます。

## 🛠️ 開発者向け（ソースからビルド）

```bash
git clone https://github.com/yousayrock/013-pocketpad.git
cd 013-pocketpad
```

**PC側（要 .NET 8 SDK）**

```bash
cd pc/PocketPadTray
dotnet run                      # そのまま起動
# 配布用の単一EXE（.NET不要版）を作る場合（圧縮なしだと約172MBになるので注意）：
dotnet publish -c Release -r win-x64 --self-contained -p:PublishSingleFile=true -p:EnableCompressionInSingleFile=true
```

**スマホ側（要 Flutter SDK）**

```bash
cd app
flutter pub get
flutter build apk --release    # APK作成
flutter run                     # USB接続した実機で直接起動
```

> ⚠️ IPv6オンリー回線（一部のモバイル回線・テザリング）でGradleがタイムアウトする場合は、環境変数 `GRADLE_OPTS=-Djava.net.preferIPv6Addresses=true` を設定してください。本リポジトリの `gradle.properties` には設定済みです。

## 🏗️ アーキテクチャ

```
┌──────────────────┐
│ Android          │  Flutter
│  PocketPad App   │
└────────┬─────────┘
         │ WebSocket :9013（LAN内のみ）
         │  テキストフレーム＝JSON制御（入力・設定同期）
         │  バイナリフレーム＝マウス移動/スクロール（5バイト・8ms間引き）
┌────────▼─────────┐
│ Windows          │  .NET 8 常駐トレイアプリ
│  PocketPadTray   │  Kestrel WebSocket + SendInput（P/Invoke）
│   ├ settings.json│  設定の正（%APPDATA%\PocketPad、双方向同期）
│   └ :9013/ HTTP  │  設定ダッシュボード（localhost限定）
└──────────────────┘
```

プロトコルの詳細は [docs/protocol.md](docs/protocol.md) を参照。
